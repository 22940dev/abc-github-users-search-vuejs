# vue-github

Search github users and repositories using their own API and VueJS.

## Project setup
```sh
$ npm install
```

### Compiles and hot-reloads for development
```sh
$ npm run serve
```

### Compiles and minifies for production
```sh
$ npm run build
```

### Run your tests

```sh
$ npm run test
```
